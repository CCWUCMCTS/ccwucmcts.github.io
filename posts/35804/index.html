/*带的树状数组板子发现是坏的->_->

然后从左到右算改了N发还是不对。。。*/

emmmm没有啥树状数组，从左到右扫一遍看成爬山，用一个数组cnt[i]来统计在i位置向上还是向下，对于每组l，r，cnt[l]++，cnt[r+1]--，从0扫到max(r)+1，对于每个cnt[i]不为0的i，用i和上一个cnt不为0的位置和当前层数更新ans数组，然后更新当前层数。

```
 1 #include<iostream>
 2 #include<cstdio>
 3 using namespace std;
 4 int n,cnt[1000020],last,now,rmx,ans[1000020]; 
 5 int main(){
 6     cin>>n;
 7     int l,r;
 8     for(int i=1;i<=n;i++){
 9         scanf("%d%d",&l,&r);
10         rmx=max(rmx,r);
11         cnt[l]++;
12         cnt[r+1]--;
13     }
14     for(int i=0;i<=rmx+1;i++){
15         if(cnt[i]!=0){
16             ans[now]+=(i-last);
17             last=i;
18             now+=cnt[i];
19         }
20     } 
21     cout<<ans[1];
22     for(int i=2;i<=n;i++){
23         printf(" %d",ans[i]);
24     }
25     return 0;
26 }```
