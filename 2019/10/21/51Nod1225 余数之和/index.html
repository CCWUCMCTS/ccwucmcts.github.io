51Nod1225 余数之和

Problem

F(n) = (n % 1) + (n % 2) + (n % 3) + …… (n % n)。其中%表示Mod，也就是余数。
例如F(6) = 6 % 1 + 6 % 2 + 6 % 3 + 6 % 4 + 6 % 5 + 6 % 6 = 0 + 0 + 0 + 2 + 1 + 0 = 3。
给出n，计算F(n), 由于结果很大，输出Mod 1000000007的结果即可。
